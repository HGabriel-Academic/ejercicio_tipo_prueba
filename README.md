# Álbum de Figuritas Mundial 2026

## Nombre del estudiante
Gabriel Benavides Ramirez

## Descripción breve de la app
Esta aplicación permite gestionar un álbum de figuritas del Mundial 2026 de forma visual e interactiva. El usuario puede cambiar el estado de cada figurita entre falta, tengo y repetida, buscar por nombre o número, filtrar por equipo y ver un resumen del progreso del álbum. Además, el estado se guarda automáticamente en el navegador mediante localStorage.

## Instrucciones para ejecutar el proyecto localmente
1. Asegúrate de tener Node.js instalado en tu equipo.
2. Abre una terminal en la carpeta del proyecto.
3. Instala las dependencias:
   ```bash
   npm install
   ```
4. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```
5. Abre la URL que aparezca en la terminal, por ejemplo: http://127.0.0.1:3002/
